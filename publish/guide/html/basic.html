1. CSS

작성 순서

display / overflow / visibility / position(left, top) / z-index / float(clear) /
width / height / margin / padding /
background(color, image, position, repeat) / border /
text-align(vertical-align) / color /
font(weight, size, line-height) / text-overflow / text-decoration /
zoom / content / other

이름 규칙 (class)

- 직관적이고 누구나 알아볼 수 있으며 간단하게 작성하는 것이 좋습니다.
- 소문자만 사용하며 단어 구분은 "-"로 표현합니다.
- 레이아웃을 구성하는 내용만 id(wrapper, header, container, nav, content, footer) 값으로 넣었습니다.

띄어쓰기

- CSS에서의 띄어쓰기는 모두 1 스페이스로 합니다.
