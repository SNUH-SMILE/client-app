morpheus-reset.min.css

morpheus-reset

morpheus-reset.min.css 파일은 수정이 없어야 하기 때문에 min파일로 압축하여 배포합니다.
모바일과 웹에 모두 사용 가능합니다.

Default 적용 내용

- 뷰포트 변환시 자동확대 방지

- box-sizing 적용

- margin, padding 0 적용


Link

- text-decoration 제거

- outline 제거


Typography

- hgroup : font weight normal 및 size 상속

- strong : font weight normal

- 그와 vertical align middle 적용


List

- list style 제거


Forms

- button cursor 적용

- border, outline 제거, bg color whit적용

- textarea vetical align top 적용 및 border 적용


Table

- caption, legend display 제거

- border-collapes collapse 적용 및 빈 셀 보이기, border-spacing 제거

- table-layout fixed 적용


input

- placeholder color #666 적용

- search, number 초기화

공통 클래스 ; Align

ta-r / right : 우정렬

al-l / left : 좌정렬

ta-c / center : 중앙정렬

va-t : 위쪽 정렬

va-m : 중간 정렬

va-b : 아래쪽 정렬

fl-l : display inline-block 변경 후 좌측 float

fl-r : display inline-block 변경 후 우측 float

over_-h : overflow hidden 적용

clear / clear-l / clear-r : both / left / right float clear

posi-a : position absolute 적용

posi-r : position relativeㄷ 적용

공통 클래스 ; Display

block : display block 적용

inline : display inline 적용

공통 클래스 ; Font weight

normal : 기본 굵기로

bold : 굵게

공통 클래스 ; Hidden

blind : 위치 및 높이값 유지하며 내용만 삭제

none : display 자체를 삭제

공통 클래스 ; Text overflow with ellipsis

ellipsis : 한줄 줄임말

ellipsis2 : 두줄 줄임말

공통 클래스 ; word-break

break : 영역을 벗어나는 텍스트 줄 바꿈 처리(예:긴 URL)

공통 클래스 ; webkit-box

# Android 4.4 이상에서만 적용이 가능합니다.

hbox : row webkit box 적용

vbox : column webkit box 적용

hbox, vbox 는 -webkit / -moz / -ms 가 모두 적용되어 있습니다.

hbox, vbox 의 바로 아래 자식은 display가 block 적용됩니다.


ss : 가로/세로 중앙 정렬

es : 가로 끝 정렬 / 세로 중앙 정렬

sc : 가로 시작 정렬 / 세로 중앙 정렬

ee : 가로 끝 정렬 / 세로 끝 정렬

cc : 가로 중앙 정렬 / 세로 중앙 정렬

js : 가로 justify 정렬 / 세로 시작 정렬

jc : 가로 justify 정렬 / 세로 중앙 정렬

je : 가로 justify 정렬 / 세로 끝 정렬

stretch : 가로/세로 justify 정렬 및 높이 채움


flex : 동일한 사이즈로 분배/적용

flex 의 바로 아래 자식은 동일한 속성을 가집니다.

flex2 / flex3 / flex4 / flex5 / flex6: 동일한 사이즈로 분배 후 클래스명에 기입된 숫자만큼 제곱됩니다. (flex 2 : X2)

예시

hbox ss

hbox cc

hbox je


hbox stretch - flex

hbox stretch - flex

hbox stretch - flex

hbox stretch - flex

hbox stretch - flex4

hbox stretch - flex
