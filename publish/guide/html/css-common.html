morpheus-common.css

morpheus-common

html 파일에 morpheus-style 또는 morpheus-common을 link 걸어서 사용을합니다.
<link rel="stylesheet" type="text/css" href="../../css/morpheus-style.css"/>

morpheus-style과 morpheus-common를 분리를 하는 이유는 다음과 같습니다.

- 2인이상 프로젝트를 작업하는 경우
- 공통/상세 스타일 적용 분리가 필요한 경우

소형프로젝트 또는 굳이 분리가 필요하지 않은 경우 morpheus-style, morpheus-common 두개의 파일을 하나로 합쳐서 사용하셔도 무관합니다.

이 파일과 함께 배포된 morpheus-common에는 guide template에 해당하는 코드가 함께 들어있으니 참고해 주시기 바랍니다.
